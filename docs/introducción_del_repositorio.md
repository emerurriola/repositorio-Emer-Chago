# presentación del proyecto

#### integrantes
* Santiago Alcázar
* Emerson Urriola 


*El objetivo de nuestro proyecto es determinar de qué manera ha funcionado el sistema proporcional en las elecciones parlamentarias de Chile, a la hora de generar mayor representatividad en el Congreso. Esto, en contraste con el sistema binominal implementado anteriormente. ¿El sistema proporcional ha servido para eliminar la hegemonía de las grandes coaliciones políticas de izquierda y derecha?.*
